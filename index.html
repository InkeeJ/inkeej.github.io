Selected Projects

  • Predicting Aviation Accident Severity - The Erdős Institute - Summer 2025
    GitHub | Video | Slides
    Led data extraction and cleaning by converting .mdb files to .csv and integrating event, aircraft, and sub-aircraft tables into a unified dataset. Engineered and imputed features for classification and regression tasks. Built ensemble models (XGBoost, Random Forest, Extra Trees, Histogram Gradient Boosting) with grid search and cross-validation. Achieved an F1 score of 0.459 (Extra Trees) in aircraft damage classification, significantly above the 0.316 baseline.
  • Logifold Implementation for adversarial attacks on Deep Neural Networks - Spring 2025 - Present
    On-going project developing a geometric framework for adversarial attacks using Logifold structures to detect perturbations, enhance robustness, and improve ensemble performance.
    Includes generation of adversarial CIFAR-10 datasets and neural network models in TensorFlow 2, with evaluations on detection strategies.
  • Persistent Laplacian on Graph Neural Networks - Spring 2025 - Present
    On-going project exploring the application of persistent Laplacian in graph neural networks.
